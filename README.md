---

Installation
---

1. Download this repo and unzip.
2. Chrome.app
3. Window > Extensions
4. Toggle `Developer mode`
5. `Load unpacked` extension.
6. Select the `TCI-night-mode` folder you've downloaded.
7. Enjoy!
